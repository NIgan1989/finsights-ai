# 🚀 FinSights AI - Быстрый старт

## Что нового?

FinSights AI теперь включает:
- 📊 **Продвинутую финансовую аналитику** с 20+ коэффициентами
- 📈 **Интерактивные графики** с зумом и экспортом
- 🎯 **Систему управления рисками** с автоматическими уведомлениями
- 📋 **Многоформатный экспорт** (CSV, JSON, Excel, PDF)
- 🎨 **Современный дизайн** корпоративного уровня

## Быстрый запуск

```bash
# 1. Установите зависимости
npm install --legacy-peer-deps

# 2. Запустите приложение
npm start

# 3. Откройте в браузере
# http://localhost:3000
```

## Как использовать

1. **Загрузите PDF выписку** - поддерживаются Kaspi, Halyk и другие банки
2. **Изучите дашборд** - 6 вкладок с разными видами анализа:
   - 📈 Обзор - основные KPI и тренды
   - 💰 Денежные потоки - операционная эффективность  
   - 📊 Категории - структура расходов
   - 🧮 Коэффициенты - все financial ratios
   - 🔮 Прогнозы - предсказательная аналитика
   - ⚠️ Риски - система раннего предупреждения

3. **Экспортируйте отчеты** - выберите нужный формат из меню графика

## Новые возможности

### 💳 Улучшенные KPI карточки
- Сравнение с предыдущими периодами
- Прогресс-бары с целевыми значениями
- Анимированные счетчики

### 📊 Интерактивные графики
- Переключение типов "на лету"
- Зум и полноэкранный режим
- Автоматические статистики

### 🧮 Профессиональная аналитика
- **Рентабельность**: ROA, ROE, ROIC, маржи
- **Ликвидность**: текущая, быстрая, абсолютная
- **Эффективность**: оборачиваемость активов, цикл конвертации
- **Риски**: многофакторная оценка с рекомендациями

### 📄 Расширенный экспорт
- **CSV**: структурированные данные с KPI
- **JSON**: API-ready формат с метаданными
- **Excel**: многолистовая структура
- **PDF**: профессиональные отчеты с графиками

## Технические характеристики

- **Frontend**: React 18 + TypeScript

- **Charts**: Recharts с кастомными компонентами
- **Animations**: Framer Motion (опционально)
