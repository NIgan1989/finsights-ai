# Анализ структуры проекта FinSights AI

## Общее описание

**FinSights AI** — это современное React-приложение для финансового анализа и учета, предназначенное для малого и среднего бизнеса. Приложение позволяет загружать финансовые данные, автоматически их категоризировать с помощью ИИ и генерировать профессиональные финансовые отчеты в соответствии с международными стандартами (GAAP/МСФО).

## Технологический стек

### Основные технологии
- **React 18.2.0** - основная библиотека для пользовательского интерфейса
- **TypeScript 4.9.5** - строгая типизация для надежности кода
- **Material-UI 5.15.21** - современная система дизайна и компонентов
- **React Router 6.23.1** - маршрутизация (пока не используется активно)

### Библиотеки для работы с данными
- **Recharts 2.15.4** - библиотека для создания интерактивных графиков
- **@mui/x-data-grid 7.9.0** - продвинутая таблица для работы с данными
- **DayJS 1.11.11** - работа с датами
- **@react-pdf/renderer 4.3.0** - генерация PDF-отчетов

### ИИ интеграции
- **@google/generative-ai 0.14.1** - интеграция с Google Gemini API
- Поддержка OpenAI API для обработки изображений и текста

### Интернационализация
- **i18next 25.3.0** - система локализации
- Поддержка английского, русского и казахского языков

### Дополнительные инструменты
- **TailwindCSS 3.4.3** - утилитарный CSS-фреймворк
- **html2canvas 1.4.1** - создание скриншотов для PDF
- **jsPDF 2.5.1** - дополнительная библиотека для PDF

## Архитектура проекта

### Структура папок

```
src/
├── components/          # React компоненты
├── contexts/           # React контексты для глобального состояния
├── services/           # Бизнес-логика и интеграции
├── utils/              # Вспомогательные функции
├── locales/            # Файлы переводов
├── types.ts            # TypeScript типы
├── constants.ts        # Константы приложения
├── theme.ts            # Настройки темы Material-UI
└── i18n.ts            # Конфигурация интернационализации
```

### Основные компоненты

#### 1. **App.tsx** - Главный компонент
- Управляет общим состоянием приложения
- Содержит логику загрузки и обработки файлов
- Переключение между дашбордом и AI-ассистентом
- Управление темами (светлая/темная)
- Переключение языков

#### 2. **Dashboard.tsx** - Основная панель аналитики
- Отображение KPI и ключевых метрик
- Интерактивные графики доходов/расходов
- Анализ денежных потоков
- Налоговые расчеты
- Бизнес-аналитика

#### 3. **DataUpload.tsx** - Компонент загрузки данных
- Поддержка drag & drop
- Обработка CSV, PDF и изображений
- Валидация файлов

#### 4. **AiAssistant.tsx** - ИИ-помощник
- Чат-интерфейс для вопросов по финансам
- Анализ данных с помощью ИИ
- Рекомендации по оптимизации

### Сервисы (Бизнес-логика)

#### 1. **financeService.ts** - Основной сервис обработки финансов
- **Обработка файлов**: CSV, PDF, изображения
- **Категоризация транзакций** по международным стандартам
- **Генерация финансовых отчетов**:
  - Отчет о прибылях и убытках (P&L)
  - Отчет о движении денежных средств (Cash Flow)
  - Балансовый отчет (Balance Sheet)
- **Расчет финансовых коэффициентов**
- **Амортизация и капитализация активов**

#### 2. **geminiService.ts** - Интеграция с Google Gemini
- Категоризация транзакций
- Генерация финансовых отчетов
- Обработка пользовательских запросов
- Получение уточнений по транзакциям

#### 3. **openaiService.ts** - Интеграция с OpenAI
- Извлечение данных из изображений
- Продвинутая категоризация расходов
- Выявление транзакций, требующих уточнения

#### 4. **bankPdfService.ts** - Специализированный парсер банковских выписок
- Обработка PDF-выписок различных банков
- Валидация банковских данных
- Форматирование результатов парсинга

#### 5. **pdfService.tsx** - Генерация PDF-отчетов
- Создание профессиональных финансовых отчетов
- Поддержка графиков и таблиц в PDF
- Мультиязычная поддержка в отчетах

### Типы данных (types.ts)

#### **Transaction** - Базовая модель транзакции
```typescript
{
  id: string;
  date: string;
  description: string;
  amount: number;
  type: 'income' | 'expense';
  category: string;
  transactionType: 'operating' | 'investing' | 'financing';
  isCapitalized: boolean;
  needsClarification?: boolean;
}
```

#### **FinancialReport** - Структура финансового отчета
- **PnLData** - Отчет о прибылях и убытках
- **CashFlowData** - Отчет о движении денежных средств  
- **BalanceSheetData** - Балансовый отчет

### Контексты

#### 1. **ThemeContext.tsx** - Управление темами
- Переключение между светлой и темной темами
- Сохранение предпочтений пользователя

#### 2. **DateRangeContext.tsx** - Управление периодами
- Фильтрация данных по датам
- Глобальное состояние выбранного периода

## Особенности и функциональность

### 1. **Многоязычность**
- Поддержка 3 языков: английский, русский, казахский
- Автоматическое определение языка браузера
- Локализация всех текстов и отчетов

### 2. **Обработка данных**
- **CSV файлы**: прямой парсинг с поддержкой различных форматов дат
- **PDF выписки**: специализированный парсер для банковских документов
- **Изображения**: извлечение данных с помощью OCR и ИИ
- **Автоматическая категоризация** транзакций

### 3. **Финансовая аналитика**
- Соответствие международным стандартам (GAAP/МСФО)
- Расчет 15+ финансовых коэффициентов
- Анализ трендов и динамики
- Прогнозирование и рекомендации

### 4. **Налоговый анализ**
- Расчеты по налогам Казахстана (НДС, КПН, социальный налог)
- Налоговый календарь
- Рекомендации по оптимизации

### 5. **Визуализация данных**
- Интерактивные графики с Recharts
- Адаптивный дизайн для всех устройств
- Экспорт графиков в PDF

### 6. **ИИ-возможности**
- Умная категоризация транзакций
- Чат-ассистент для финансовых вопросов
- Автоматическое выявление аномалий
- Персонализированные рекомендации

## Категории транзакций

### Расходы (соответствуют международным стандартам):
- **Себестоимость**: Закупка товаров, сырье, производственная зарплата
- **Операционные расходы**: Маркетинг, административные расходы, коммунальные услуги
- **Финансовые расходы**: Проценты по кредитам, курсовые разницы
- **Капитальные затраты**: Оборудование, недвижимость, ПО
- **Прочие расходы**: Налоги, штрафы, представительские расходы

### Доходы:
- **Операционные**: Основная выручка, дополнительные услуги
- **Финансовые**: Процентные доходы, курсовые разницы
- **Инвестиционные**: Продажа активов
- **Прочие**: Получение кредитов, взносы учредителей

## Настройки и конфигурация

### Переменные окружения
- `REACT_APP_GEMINI_API_KEY` - ключ API для Google Gemini
- `REACT_APP_OPENAI_API_KEY` - ключ API для OpenAI (опционально)

- `npm start` - запуск в режиме разработки
- `npm run build` - сборка для продакшена
- `npm run setup-local` - настройка локального окружения
