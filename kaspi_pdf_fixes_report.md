# 🔧 Исправления парсера PDF выписок банка Каспи

## 📋 Обнаруженные проблемы

1. **❌ PDF.js не подключен** - Отсутствовала библиотека для чтения PDF файлов
2. **❌ Неточные регулярные выражения** - Парсер не распознавал различные форматы транзакций
3. **❌ Некорректная обработка сумм** - Проблемы с парсингом чисел с пробелами и разделителями
4. **❌ Неправильное определение типа транзакции** - Доходы/расходы определялись неточно
5. **❌ Отсутствие отладочной информации** - Сложно было понять, почему парсинг не работает

```javascript
const patterns = [

  /^(\d{1,2}\.\d{1,2}\.\d{4})\s+(.+?)\s+([\+\-]?\d[\d\s]*[,.]?\d{0,2})\s*₸?\s*$/,
```

### 3. 💰 Улучшенная обработка сумм

```javascript
// Очистка суммы от лишних символов
let cleanAmountStr = amountStr
  .replace(/\s+/g, '') // Убираем пробелы
  .replace(/[^\d\+\-,\.]/g, '') // Оставляем только цифры, знаки и разделители
  .replace(/,/g, '.'); // Заменяем запятые на точки

// Определение знака
const isNegative = cleanAmountStr.includes('-');
const isPositive = cleanAmountStr.includes('+');
cleanAmountStr = cleanAmountStr.replace(/[\+\-]/g, '');

const amount = parseFloat(cleanAmountStr);
```

```javascript
// Сортировка элементов по позиции для правильного порядка чтения
const pageText = textContent.items
  .sort((a: any, b: any) => {
    // Сортируем по вертикальной позиции (y), затем по горизонтальной (x)
    if (Math.abs(a.transform[5] - b.transform[5]) > 2) {
      return b.transform[5] - a.transform[5]; // Сортировка сверху вниз
    }
    return a.transform[4] - b.transform[4]; // Сортировка слева направо
  })
  .map((item: any) => item.str)
  .join(' ');
```

## 🎯 Результат

### Поддерживаемые форматы выписок Каспи:

1. **Базовый формат**: `12.03.2024 Описание транзакции 1500.00 ₸`
2. **С временем**: `12.03.2024 14:30 Описание транзакции +1500.00 ₸`
3. **Без валюты**: `12.03.2024 Описание транзакции -1500.00`
4. **С разделителями**: `12.03.2024 Описание транзакции 1 500,00 ₸`
5. **Табличный формат**: Строки с колонками, разделенными пробелами

### Обрабатываемые типы транзакций:

✅ **Доходы автоматически определяются по:**
- Знаку `+` в сумме
- Ключевым словам: "пополнение", "поступление", "зарплата", "перевод на счет", "возврат", "доход"

✅ **Расходы определяются по:**
- Знаку `-` в сумме
- Все остальные транзакции по умолчанию

### Форматы дат:
- `ДД.ММ.ГГГГ` (12.03.2024)
- `Д.М.ГГГГ` (2.3.2024)
- `ДД.ММ.ГГ` (12.03.24) - автоматически преобразуется в 20ГГ

### Форматы сумм:
- `1500.00` - точка как разделитель
- `1500,00` - запятая как разделитель  
- `1 500.00` - с пробелами
- `+1500.00` / `-1500.00` - с явным знаком
- `1500.00 ₸` - с символом валюты

## 🚀 Как использовать

1. **Загрузите PDF выписку** из Каспи банка в приложение
2. **Откройте консоль браузера** (F12) для просмотра детальной отладочной информации
3. **Проверьте результат** парсинга в таблице транзакций
4. **При проблемах** - посмотрите в консоли, какие строки не были распознаны
